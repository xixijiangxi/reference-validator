# 如何分享应用给他人使用

本应用是一个本地 Web 应用，要让其他人也能使用，有以下几种方式：

## 🚀 方式一：使用 ngrok 内网穿透（推荐 - 最简单）

**适用场景**：快速分享给任何人，无论他们在哪里

### 步骤：

1. **安装 ngrok**
   - 访问 https://ngrok.com/download
   - 下载 Windows 版本并解压
   - 注册账号（免费）并获取 authtoken
   - 在命令行运行：`ngrok config add-authtoken YOUR_TOKEN`

2. **启动应用**
   ```bash
   start_with_ngrok.bat
   ```
   这个脚本会：
   - 启动后端服务（允许外部访问）
   - 启动前端服务
   - 自动启动 ngrok 隧道

3. **获取分享链接**
   - 查看新打开的 ngrok 窗口
   - 会显示类似这样的信息：
     ```
     Forwarding   https://xxxx-xxx-xxx-xxx.ngrok-free.app -> http://localhost:5173
     ```
   - 复制 `https://xxxx-xxx-xxx-xxx.ngrok-free.app` 这个地址

4. **分享给他人**
   - 将 ngrok 生成的链接分享给其他人
   - 他们可以直接在浏览器中打开使用

**注意**：
- 免费版 ngrok 链接每次重启都会变化
- 需要你的电脑保持运行
- 免费版有连接数限制

---

## 🏠 方式二：局域网访问（同一 WiFi）

**适用场景**：其他人跟你连接在同一个 WiFi/局域网

### 步骤：

1. **启动应用**
   ```bash
   start_lan.bat
   ```

2. **获取你的 IP 地址**
   - 脚本会自动显示你的 IP 地址
   - 例如：`http://192.168.1.100:5173`

3. **分享地址**
   - 将显示的 IP 地址分享给同一网络下的其他人
   - 例如：`http://192.168.1.100:5173`

**注意**：
- 需要确保防火墙允许端口 5173 和 8000
- 其他人必须连接在同一个 WiFi/局域网

---

## ☁️ 方式三：部署到云服务器（长期使用）

**适用场景**：想要长期分享，不需要你的电脑一直运行

### 推荐平台：

1. **Railway**（简单，有免费额度）
   - 访问 https://railway.app/
   - 使用 GitHub 账号登录
   - 连接你的代码仓库
   - 自动部署

2. **Render**（免费，简单）
   - 访问 https://render.com/
   - 使用 GitHub 账号登录
   - 创建 Web Service
   - 连接代码仓库

详细步骤请查看 [DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md)

## ⚠️ 重要提示

1. **API Key 安全**
   - 不要将你的 DashScope API Key 分享给他人
   - 如果部署到云服务器，使用环境变量配置

2. **防火墙设置**
   - 如果使用局域网访问，确保 Windows 防火墙允许端口 5173 和 8000
   - 可以在 Windows 防火墙中添加规则

3. **网络连接**
   - 确保你的网络连接稳定
   - ngrok 需要你的电脑保持运行

## 🆘 常见问题

**Q: ngrok 链接访问时显示错误？**
A: 确保本地应用已启动，检查 ngrok 窗口是否有错误信息。

**Q: 其他人无法访问局域网地址？**
A: 
- 检查是否在同一 WiFi
- 检查防火墙设置
- 尝试关闭防火墙测试

**Q: 如何固定 ngrok 域名？**
A: 需要升级到 ngrok 付费版，或使用 frp 自建服务器。

---

## 🎯 推荐流程

**第一次分享（快速测试）：**
1. 运行 `start_with_ngrok.bat`
2. 复制 ngrok 显示的链接
3. 分享给他人

**长期使用：**
1. 部署到 Railway 或 Render
2. 获得固定域名
3. 分享域名给他人
